# 전국 산업공학 대학생 경진대회 (2024 Field Camp)

## Overview
전국 산업공학과 학생을 대상으로 한 학술 교류 캠프 내 경진대회에서 문제 설계 및 평가위원으로 활동하였습니다.

- 2024 FIELD CAMP
- 대한산업공학회 산하 학술 동아리 FIELD(Future Industrial Engineering Leaders and Dreamers)에서 주관
- FIELD 동아리 내 학술교류부서인 Competition 부서로 활동하며 캠프 내 경진대회를 담당

## Activity Period

- 2024.03 – 2024.08  
- 캠프 진행 기간: 2024.08.07 – 2024.08.09  
- 참가자: 164명  
- 스태프: 54명

## Role & Responsibilities
- 경진대회 문제 설계
- 실제 데이터 기반 시뮬레이션 시나리오 구축
- 평가 기준 수립
- 참가자 답안 평가 및 시상 주관

  
## Competition Theme
### 시뮬레이션 툴을 활용한 개찰구 시스템 최적화 (Traffic Facility Optimization Using Simulation Tools)

대회 참가자들이 수행한 문제 과정: 
1. 실제 지하철 승객 데이터 분석
2. 도착 분포 추정 및 파라미터 산정
3. 시뮬레이션 모델 구현
4. 개찰구 시스템 최적화 방안 도출


## Data Collection & Analysis
### 1) 데이터 수집

- 서울 지하철 삼성역 승객 데이터 활용
  대회 참가자들이 비정제 데이터를 직접 전처리하여 사용할 수 있도록 문제 설. 따라서 원시 데이터 수집을 위해 직접 삼성역에서 승객 데이터 수집
- 1시간 기준 승·하차 인원 데이터 분석(시간 당 개찰구 통과 인원, 각 인원 별 소요 시간 측정)


### 2) 파이썬 기반 통계 분석

- 원시 데이터 분포 분석
- 지수분포 가정 및 파라미터 추정
- 도착률(arrival rate) 계산
- 분포 가정의 타당성 검토



## Simulation Modeling
### 사용 툴: AnyLogic

- 보행자 속도 및 크기 설정
- 서비스 로직 구현
- 1500초 시뮬레이션 수행
- 병목 구간 분석
- 최적 개찰구 수 도출
